Input Controls

Wijmo includes nineteen different types of input controls, which can be used alone, as an input field in a form, or in conjunction with one of Wijmo's other controls. Like Wijmo's other controls, these use CollectionViews to manage their data; this means that its both easy to manage the data of the control, as well as easy to mimic data changes across multiple controls.

This page will outline some of Wijmo's input components, as well as some of their key features.

AutoComplete

The AutoComplete control functions as both a dropdown and a search bar. Users can expand the dropdown via the arrow button, but they can also type into the text area and be shown a list of terms that match what they've entered.




ComboBox



The ComboBox control serves as Wijmo's basic dropdown control. It allows users to select an item from a dropdown panel, as well as type inside of the textarea and have the control filter to the value that matches what the user is typing. This control is similar to the AutoComplete control.
